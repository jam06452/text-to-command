# TTC

- Made in elixr & phoenix with a react frontend
- Just input what you want the command to do in the box and select the operating system. Then a command will appear below it to copy

## Setup

- Either clone or pull the docker image
- Include in runtime arguments the OPENAI URL, OPENAI Key and SECRET_KEY_BASE this can be made with mix phx.gen.secret.
- Then either run mix phx.server OR bring online with docker
